# tc3005b-movil

Mobile port of the web project into Expo / React Native.

## Run

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm run start
```

3. Set the backend URL in `.env` or your shell:

```bash
EXPO_PUBLIC_API_URL=https://todo-backend-88437624090.us-central1.run.app
```

Also add your Firebase config values to `.env`:

```bash
EXPO_PUBLIC_FIREBASE_API_KEY=...
EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN=...
EXPO_PUBLIC_FIREBASE_PROJECT_ID=...
EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=...
EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=...
EXPO_PUBLIC_FIREBASE_APP_ID=...
```
